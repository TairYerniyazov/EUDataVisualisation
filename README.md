# EU Data Visualisation
> Maps showing life expectancy in the EU countries.
<br> Languages & Libraries used: Python, geopandas, Beautiful Soup, matplotlib. 
<br> Platform: Google Colab.
<br> Alternative way to access the Colab notebook: <a src="https://colab.research.google.com/drive/13AIqlPC5Qxe5_B5SuXpffCLLPc4-HrNB?usp=sharing">life_expectancy.ipynb</a>. 

## Description
I wanted to know an easy way to generate maps using some amount of data given. It seems to be a little hard to find something like that on the web as most of the examples available use the existing methods of visualising statistical data the application of the dedicated libraries which do not provide tools for find-tuning specific regions. I use data related to the life expectancy in the EU as an example. Do not hesitate to make changes and apply your input to generate something similar to the examples shown below.
